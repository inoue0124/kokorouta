# 要件定義書: Settings（設定画面）

> 生成日時: 2026-02-27
> ステータス: Draft

## 1. 概要

アプリ設定・アカウント管理画面。ブロックリスト管理、アカウント削除機能を含む。App Store 審査要件（UGC アプリのブロック機能、アカウント削除機能）を満たす。

## 2. スコープ

### 含むもの

- 設定画面（S-005）
- ブロックリスト画面（S-009）
- アカウント削除画面（S-008）
- 設定タブ内のナビゲーション

### 含まないもの

- 通知リマインダーの実装（UI のみ。実際のプッシュ通知は別フィーチャー）
- 利用規約・プライバシーポリシーの URL（プレースホルダー）

## 3. 機能要件

### FR-001: 設定一覧

- ブロックリストへの遷移
- アカウント削除への遷移
- アプリバージョン表示

### FR-002: ブロックリスト

- ブロック中のユーザー一覧を表示
- 各ユーザーに「解除」ボタンを表示
- 解除タップでブロック解除 API を呼び、リストから除去

### FR-003: アカウント削除

- 確認テキスト入力（「削除」と入力で有効化）
- 確認テキスト一致でアカウント削除 API を呼ぶ
- 削除完了後の状態をハンドリングする

## 4. 受け入れ条件

- [ ] 設定画面が表示される
- [ ] ブロックリスト画面でブロック中ユーザーが一覧される
- [ ] ブロック解除が動作する
- [ ] アカウント削除画面で「削除」入力後にボタンが有効化される
- [ ] アカウント削除 API が呼ばれる
- [ ] ビルドが通る
