# 要件定義書: MyTanka（マイ短歌履歴）

> 生成日時: 2026-02-27
> ステータス: Draft

## 1. 概要

自分が作成した短歌の履歴を一覧表示する画面。フィードと同じ `TankaCard` コンポーネントを使用するが、いいねボタンは非表示（自分の短歌）。

## 2. スコープ

### 含むもの

- マイ短歌履歴画面（S-004）
- マイ短歌タブ内のナビゲーション

### 含まないもの

- 短歌の編集・削除機能（スコープ外）
- 他ユーザーの短歌（Feed フィーチャー）

## 3. 機能要件

### FR-001: マイ短歌一覧表示

- 起動時に自分の短歌を取得し、`TankaCard` で一覧表示する
- いいねボタンは非表示（`onLike` を nil にする）
- いいね数は表示しない

### FR-002: プルリフレッシュ

- 下方向にプルしてデータを最新に更新する

### FR-003: 状態表示

- ローディング中: `LoadingView`
- エラー: `ErrorView` + リトライ
- 空データ: `EmptyStateView("まだ短歌がありません。今日の悩みを詠んでみましょう")`

## 4. 受け入れ条件

- [ ] マイ短歌画面が表示され、自分の短歌カードが一覧される
- [ ] カードタップでフリップアニメーションが動作する
- [ ] プルリフレッシュでデータが更新される
- [ ] 空データ状態が正しく表示される
- [ ] ビルドが通る
